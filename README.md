# The first Rush of the C++ bootcamp

> A retro style shoot 'em up'.
>
> A kin to Atari games, created using the ncurses library.
>

## Installation

iOS X & Linux:

```sh
git clone https://github.com/Kid-Seven-7/Cpp_Rush_00.git
make all
```

Windows:

```sh
Not sure. Sorry
```

## Usage example

After cloning and making typing
``./ft_Retro``
will begin the game.
